# TravelTide Customer Insight: Customer Segmentation and Rewards Program
TravelTide MSIT Project
This project's aim is to assist the TravelTide marketing team in creating a rewards program with deeper insight into their userbase.

# Project Overview
In this project we are taking a closer look at the users of TravelTide, an e-booking startup.
We have been tasked with analysing userdata in order to help the team with its rewards program.

# Project Key Features
- Exploratory Data Analysis: insights into customer demographic, behaviour, preferences, and booking patterns.

- K-Means Clustering: applied K-Means clustering to segment customers into distinct groups.

- Optimal Cluster Count: determined the optimal number of clusters using the Elbow Plot and Silhouette Score.

- Perk Insights: analysis revealed which perks are most likely to appeal to specific customer segments.

- Recommendations: provide recommendations for TravelTide, including the possibility of a multi-level rewards program and perk combination.
